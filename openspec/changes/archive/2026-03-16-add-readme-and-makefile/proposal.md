## Why

项目缺少中文 README 文档和 Makefile，新开发者无法快速了解项目用途和常用操作命令。需要提供标准的项目入口文档和便捷的命令封装。

## What Changes

- 新增中文 `README.md`，包含项目简介、快速开始、项目结构、常用命令说明
- 新增 `Makefile`，封装常用开发命令（安装依赖、启动服务、代码检查、格式化、测试）

## Capabilities

### New Capabilities
- `project-docs`: 中文 README 文档和 Makefile 开发命令封装

### Modified Capabilities
<!-- None -->

## Impact

- **New files**: `README.md`, `Makefile`
- **No code changes**: 仅新增文档和构建辅助文件
